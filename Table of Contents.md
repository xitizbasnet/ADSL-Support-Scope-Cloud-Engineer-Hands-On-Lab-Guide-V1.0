## 📘 Lab Overview

This Microsoft 365 Administration Lab Manual is designed to provide hands-on practice with core Microsoft 365 services and administrative tasks.

The labs cover:

* 👤 Identity and access management
* 📧 Exchange Online administration
* 💬 Microsoft Teams configuration
* 💻 Endpoint and application management using Microsoft Intune
* ☁️ OneDrive administration and policy management
* 🔐 Data Loss Prevention and compliance management
* 🛡️ Conditional Access, MFA, SSO, and directory synchronization
* 🔎 eDiscovery and data governance activities
* 📮 On-premises SMTP server management
* 🧑‍💻 End-user support operations
* 📂 SharePoint Online administration

> **Note**
> Each lab should be completed according to organizational requirements, security standards, and Microsoft 365 administrative best practices.
